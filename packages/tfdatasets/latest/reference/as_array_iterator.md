# as_array_iterator


Convert tf_dataset to an iterator that yields R arrays.




## Description

Convert tf_dataset to an iterator that yields R arrays.





## Usage
```r
as_array_iterator(dataset)
```




## Arguments


Argument      |Description
------------- |----------------
dataset | A tensorflow dataset





## Value

An iterable. Use `iterate()` or `iter_next()` to access values from the iterator.
