# dataset_concatenate


Creates a dataset by concatenating given dataset with this dataset.




## Description

Creates a dataset by concatenating given dataset with this dataset.





## Usage
```r
dataset_concatenate(dataset, ...)
```




## Arguments


Argument      |Description
------------- |----------------
dataset, ... | ``tf_dataset``s to be concatenated





## Value

A dataset
