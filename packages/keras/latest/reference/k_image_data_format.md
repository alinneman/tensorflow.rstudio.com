# k_image_data_format


Default image data format convention ('channels_first' or 'channels_last').




## Description

Default image data format convention ('channels_first' or 'channels_last').





## Usage
```r
k_image_data_format()

k_set_image_data_format(data_format)
```




## Arguments


Argument      |Description
------------- |----------------
data_format | string. ``'channels_first'`` or ``'channels_last'``.
