# k_bias_add


Adds a bias vector to a tensor.




## Description

Adds a bias vector to a tensor.





## Usage
```r
k_bias_add(x, bias, data_format = NULL)
```




## Arguments


Argument      |Description
------------- |----------------
x | Tensor or variable.
bias | Bias tensor to add.
data_format | string, ``"channels_last"`` or ``"channels_first"``.





## Value

Output tensor.
