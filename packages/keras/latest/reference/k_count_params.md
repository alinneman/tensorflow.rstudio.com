# k_count_params


Returns the static number of elements in a Keras variable or tensor.




## Description

Returns the static number of elements in a Keras variable or tensor.





## Usage
```r
k_count_params(x)
```




## Arguments


Argument      |Description
------------- |----------------
x | Keras variable or tensor.





## Value

Integer, the number of elements in ``x``, i.e., the product of the array's static dimensions.
