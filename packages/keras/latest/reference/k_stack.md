# k_stack


Stacks a list of rank ``R`` tensors into a rank ``R+1`` tensor.




## Description

Stacks a list of rank ``R`` tensors into a rank ``R+1`` tensor.





## Usage
```r
k_stack(x, axis = 1)
```




## Arguments


Argument      |Description
------------- |----------------
x | List of tensors.
axis | Axis along which to perform stacking (axis indexes are 1-based).





## Value

A tensor.
