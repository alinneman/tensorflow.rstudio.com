# k_permute_dimensions


Permutes axes in a tensor.




## Description

Permutes axes in a tensor.





## Usage
```r
k_permute_dimensions(x, pattern)
```




## Arguments


Argument      |Description
------------- |----------------
x | Tensor or variable.
pattern | A list of dimension indices, e.g. (1, 3, 2). Dimension indices are 1-based.





## Value

A tensor.
