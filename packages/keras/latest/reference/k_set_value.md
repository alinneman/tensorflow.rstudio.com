# k_set_value


Sets the value of a variable, from an R array.




## Description

Sets the value of a variable, from an R array.





## Usage
```r
k_set_value(x, value)
```




## Arguments


Argument      |Description
------------- |----------------
x | Tensor to set to a new value.
value | Value to set the tensor to, as an R array (of the same shape).
