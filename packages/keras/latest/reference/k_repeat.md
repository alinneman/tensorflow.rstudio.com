# k_repeat


Repeats a 2D tensor.




## Description

If x has shape (samples, dim) and n is 2, the output will have shape
(samples, 2, dim).





## Usage
```r
k_repeat(x, n)
```




## Arguments


Argument      |Description
------------- |----------------
x | Tensor or variable.
n | Integer, number of times to repeat.





## Value

A tensor
