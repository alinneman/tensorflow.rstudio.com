# k_gradients


Returns the gradients of ``variables`` w.r.t. ``loss``.




## Description

Returns the gradients of ``variables`` w.r.t. ``loss``.





## Usage
```r
k_gradients(loss, variables)
```




## Arguments


Argument      |Description
------------- |----------------
loss | Scalar tensor to minimize.
variables | List of variables.





## Value

A gradients tensor.
