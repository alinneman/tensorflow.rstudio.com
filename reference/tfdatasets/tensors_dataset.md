# tensors_dataset


Creates a dataset with a single element, comprising the given tensors.




## Description

Creates a dataset with a single element, comprising the given tensors.





## Usage
```r
tensors_dataset(tensors)
```




## Arguments


Argument      |Description
------------- |----------------
tensors | A nested structure of tensors.





## Value

A dataset.
