# k_batch_get_value


Returns the value of more than one tensor variable.




## Description

Returns the value of more than one tensor variable.





## Usage
```r
k_batch_get_value(ops)
```




## Arguments


Argument      |Description
------------- |----------------
ops | List of ops to evaluate.





## Value

A list of arrays.
