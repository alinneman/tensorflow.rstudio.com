# k_update_sub


Update the value of ``x`` by subtracting ``decrement``.




## Description

Update the value of ``x`` by subtracting ``decrement``.





## Usage
```r
k_update_sub(x, decrement)
```




## Arguments


Argument      |Description
------------- |----------------
x | A ``Variable``.
decrement | A tensor of same shape as ``x``.





## Value

The variable ``x`` updated.
