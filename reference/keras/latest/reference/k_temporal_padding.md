# k_temporal_padding


Pads the middle dimension of a 3D tensor.




## Description

Pads the middle dimension of a 3D tensor.





## Usage
```r
k_temporal_padding(x, padding = c(1, 1))
```




## Arguments


Argument      |Description
------------- |----------------
x | Tensor or variable.
padding | List of 2 integers, how many zeros to add at the start and end of dim 1.





## Value

A padded 3D tensor.
