# initializer_constant


Initializer that generates tensors initialized to a constant value.




## Description

Initializer that generates tensors initialized to a constant value.





## Usage
```r
initializer_constant(value = 0)
```




## Arguments


Argument      |Description
------------- |----------------
value | float; the value of the generator tensors.
