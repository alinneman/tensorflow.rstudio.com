# k_softmax


Softmax of a tensor.




## Description

Softmax of a tensor.





## Usage
```r
k_softmax(x, axis = -1)
```




## Arguments


Argument      |Description
------------- |----------------
x | A tensor or variable.
axis | The dimension softmax would be performed on. The default is -1 which indicates the last dimension.





## Value

A tensor.
