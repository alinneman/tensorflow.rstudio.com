# k_identity


Returns a tensor with the same content as the input tensor.




## Description

Returns a tensor with the same content as the input tensor.





## Usage
```r
k_identity(x, name = NULL)
```




## Arguments


Argument      |Description
------------- |----------------
x | The input tensor.
name | String, name for the variable to create.





## Value

A tensor of the same shape, type and content.
